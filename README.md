## Nicki_lottery ##
Scriptet blev requestet af @Frosty og det tog mig 2 minutter.
Hvis du har item "nicki_lottokupon" i din inventory kan du trykke
"> Skrab" og så skraber du en lottokupon og får en præmie.

## FEATURES ##
Item'et bliver selv oprettet i scriptet, så du skal ikke oprette item i vrp.

## PRÆMIER ##
Du kan være uheldig og ingenting vinde
Du kan vinde 7500 DKK
Du kan vinde 50k DKK
Du kan vinde Jackpotten på 250k DKK.
